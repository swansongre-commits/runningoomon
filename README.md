# 🏃 런닝구몬 (runningoomon)

구글 공룡게임 스타일의 러너 게임. 캐릭터(🐱 고양이 / 🐰 토끼 / 🐢 거북이 / 👽 외계인)를 골라 달리고,
스테이지 클리어마다 수학문제를 풀어 점수를 2배로 만들 수 있습니다.

## 플레이
- **정적 사이트**(GitHub Pages): `index.html` 하나로 동작 — 별도 서버 불필요
- **로컬(선택)**: `streamlit run app.py` 로도 실행 가능

## 조작
- 점프: 스페이스 / ↑ / 화면 탭(클릭)
- 숙이기: ↓ / 화면의 ⬇ 버튼
- 다시 시작: R · 게임오버 화면에선 스페이스/ESC
- 캐릭터 선택: 시작 화면에서 ← →

## 기능
- 스테이지 시스템 + 경사(오르막 느림 / 내리막 빠름)
- 스테이지 클리어 시 수학문제(정답 ×2, 오답 유지)
- 달리기 모션, 문 열림/닫힘 전환, 축하 이펙트
- **서버 공유 랭킹 TOP 10** (Supabase) — 게임오버 → 끝내기 → 이름 등록
- 모바일 반응형 + 가로모드 권장

## 랭킹 백엔드 (Supabase)
- 테이블 스키마/정책: [`supabase_setup.sql`](supabase_setup.sql)
- 클라이언트에서 publishable(anon) 키로 REST 직접 호출 (읽기/등록만 허용, RLS 적용)
